== Installation ==

### Automatic Installation (Recommended)

1. Go to your WordPress admin area
2. Navigate to `Plugins > Add New`
3. Search for "W3A11Y Artisan"
4. Click "Install Now" and then "Activate"
5. Go to `Settings > W3A11Y Artisan` to configure your API key

### Manual Installation

1. Download the plugin ZIP file
2. Go to `Plugins > Add New > Upload Plugin`
3. Choose the downloaded ZIP file and click "Install Now"
4. Activate the plugin
5. Configure your API key in plugin settings

### Configuration

1. **Get API Key**: Register at [W3A11y.com](https://w3a11y.com/register) and generate new API key at [API Keys](https://w3a11y.com/settings#api-keys)
2. **Add API Key**: Go to `W3A11Y > Artisan Settings` in your WordPress admin
3. **Test Connection**: Use the "Validate" button to verify API key and connection, then save settings
4. **Start Generating Images**: Visit Media Library and click "Generate Image With W3A11Y Artisan"
5. **Edit Images**: Select any image from Media Library and click "Edit with W3A11Y Artisan"
6. **AltText Generation**: Select any image from Media Library and click "Generate Alt Text" to get AI-generated alt text.

== Frequently Asked Questions ==

= Do I need an API key to use this plugin? =

Yes, you need a W3A11Y API key to generate and edit images and alttext. You can get a free account at [W3A11y.com](https://w3a11y.com/register) with 40 free credits included.

= What image formats are supported? =

The plugin supports JPG, PNG, and WebP formats for both generation and editing. Generated images are typically provided in high-quality PNG format. You can convert to JPG or WebP along with optimization during the save process.

= Is there a limit to how many images or alttext I can generate? =

Usage limits depend on your credits. Per image generation or editing consumes 6 credits. Per alttext generation consumes 1-2 credits based on the formats. Standard formats (jpg, png, gif and webp) cost 1 credit and Enhanced formats (svg, avif, bmp, dng, tiff, tif, heic, heif, and ico) cost 2 credits. You can monitor your credit usage during generation, plugin settings and on the [Profile Credits](https://w3a11y.com/profile/credits).

= Can I edit images that weren't generated by the AI? =

Yes! You can use the AI editing features on any image in your Media Library. Simply click the "Edit with W3A11Y Artisan" button on any image.

= Does this plugin work with page builders? =

Currently only support WordPress block editor only. Later versions we will add support for others page builders.

= What happens to my original images when editing? =

Original images are preserved. The AI editing creates a new version, so you never lose your original files. You can always revert to the original if needed.

= Can I use generated images commercially? =

Yes, images generated through W3A11Y Artisan can be used for commercial purposes according to our terms of service. Always review the latest terms for specific usage rights.

= Is technical knowledge required? =

Not at all! The plugin is designed for users of all skill levels. Simply describe what you want in plain English, and the AI handles the technical aspects of image creation and editing.

= How secure is my data? =

We prioritize security with encrypted API communications, secure data handling, and no permanent storage of your images on external servers after processing. All operations follow WordPress security best practices.

= What if I need help or support? =

We offer comprehensive support through multiple channels:
* Plugin support forum on WordPress.org
* Email support for account holders
* Documentation and tutorials at W3A11y.com
